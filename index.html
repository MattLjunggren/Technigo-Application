HTTPS

HTTPS stands for "Hypertext transfer protocol secure" and is an extension of HTML used to securely transfer data on the internet. The protocol was developed by Netscape and uses an encryption protocal called TLS/SSL to secure the data. The method starts as assymetric encryption meaning that the sender and the reciever uses two separate yet mathematically connected cryptographic keys. These keys are known as a "public key" and a "private key". When a user connects to a domain the private key is verified by the domains public key, the protocol then changes to a symmetric encryption which is faster providing both safety and short transfer time. The encryption certificates which connects the keys comes pre-installed in the browser and is provided by a third party software company.
